Documentación de Rutas de la Aplicación Flask

Esta documentación incluye todas las rutas disponibles en la aplicación, junto con una descripción y ejemplos de uso. Para más detalles tambien puede leer el README.md en la documentación para encontrar la información de como explotar las vulnerabilidades que hay en la aplicación.

Ruta Método Descripción Ejemplo de Petición Respuesta
/calcular POST Ejecuta expresiones matemáticas o código Python enviados en formato JSON. { "calculo1": "10 + 30", "calculo2": "5 * 6" } { "calculo1": 40, "calculo2": 30 }
/comentario POST Registra comentarios del usuario y puede exponer datos sensibles. [form]: "comentario" = "Esto es un comentario" html, 200
/usuarios/<usuario> GET POST Consulta información de un usuario específico o crea un nuevo usuario. GET: /usuarios/usuario1
POST: /usuarios/usuario2 		GET: html, 200
POST: {"usuario": "usuario2", "password": "pass123"}
/mensaje POST Permite enviar un mensaje al servidor. { "mensaje": "Hola mundo" } "Recibido", 200

Consideraciones de Seguridad

Algunas de las vulnerabilidades que fueron implementadas son extremadamente peligrosas en las manos incorrectas NO se recomienda ejecutar la aplicación en entornos/redes vulnerables.
